Install from ZIP

Chrome needs the extracted folder, not the ZIP itself.

  1. Click Download ZIP on this page.
  2. Extract the ZIP into a normal folder on your computer.
  3. Open chrome://extensions in Chrome.
  4. Turn on Developer mode in the top-right.
  5. Click Load unpacked.
  6. Select the extracted VeoPanel folder.
  7. Optional: pin the extension for faster access.
Important

VeoPanel is currently available as an early access manual install. Chrome does not install extensions directly from a ZIP file, so extracting first is the expected step until the Chrome Web Store release is ready.

How to Use

Open a video page and use VeoPanel as your playback layer.

  1. Open any webpage that contains an HTML5 video.
  2. Click the VeoPanel extension icon in Chrome.
  3. Use the popup to toggle the control panel on that page.
  4. Click Back 10s, Forward 10s, or Play.
  5. Set a custom jump amount if you want bigger or smaller seek steps.
  6. Use the keyboard shortcuts while watching for faster control.

Features

Focused on the controls that matter most.

Floating control panel

Shows a compact overlay on pages with an HTML5 video element.

Back and forward 10 seconds

Quick jumps for replaying lines, skipping filler, or correcting missed moments.

Play and pause

Keeps the main playback action easy to access even when the site UI is poor.

Custom jump amount

Change the skip interval instead of being locked to one fixed seek step.

Popup toggle

Enable the panel from the extension popup on the current page.

Universal HTML5 support

Designed for standard web video players across sites, including players that hide controls.
